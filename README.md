# Design Cooperation Website

A modern, responsive AI assistant landing page for Design Cooperation built with HTML, CSS, and JavaScript.

## Project structure

- `index.html` — homepage with hero, about, services, contact, and AI assistant sections
- `styles.css` — responsive styling and layout
- `script.js` — AI chat assistant behavior and quote request generation

## Run locally

1. Open `index.html` in your browser.
2. Or use a local web server:
   - Python 3: `python -m http.server 8000`
   - Node.js: `npx serve .`
3. Visit `http://localhost:8000` in your browser.

## Features

- Hero section with brand messaging
- About section and services showcase
- AI assistant chat simulator for branding and quote requests
- Quote request form collecting name, phone, and project details
- Responsive design for desktop and mobile screens
